About

Computer Graphics and Game Engine development is what I enjoy most! Real-time 3D rendering is super interesting, and seeing how modern AAA games have to run multiple render passes with thousands of objects all in under 16 ms is truly fascinating to me.

Projects

Custom Game Engine

C++ | Vulkan

Vultr is a custom 3D Game Engine built from scratch. It uses a Vulkan rendering backend with an editor that uses ImGui. It supports run-time reflection in the editor, gameplay hot-reloading, ECS gameplay system, optimized asset imports, and a full material system. The engine is capable of rendering PBR graphics with shaders for roughness, metalness, normal, and albedo. The meshes, textures, shaders, and all other assets are loaded using separate thread submission queues to maximize performance. It also uses a minimal subset of the C++ STL, opting for a custom template library where possible.

BlueBubbles

Flutter | Android

BlueBubbles is an iMessage solution for Android, Linux, and Windows that allows you to send and receive Apple iMessage texts from non-Apple devices. I started this open-source project in 2020 with some other developers and the app now has over 9,000 active users. I worked on the Android client and designed the Flutter app to communicate with our Mac server.
